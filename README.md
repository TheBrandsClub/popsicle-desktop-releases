# Popsicle Desktop — releases

Feed **solo binarios** del auto-update de [Popsicle Desktop](https://github.com/TheBrandsClub/tbc-popsicle-desktop) (The Brands Club).

`electron-updater` lee el `latest.yml` + el instalador `.exe` de cada release publicado aquí. El código fuente vive en el repo privado `tbc-popsicle-desktop`.
